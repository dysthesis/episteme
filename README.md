# Shortcuts cheatsheet

> [!tip]
>  On macOS, substitute `Ctrl` with `Cmd`.

## Templates

| Shortcut | Action                    |
| -------- | ------------------------- |
| `Ctrl-C` | Create a new lecture note |

## Navigation

| Shortcut | Action                   |
| -------- | ------------------------ |
| `Ctrl-F` | Search the entire vault. |
