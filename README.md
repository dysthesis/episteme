# Shortcuts cheat sheet

> [!tip]
>  On macOS, substitute `Ctrl` with `Cmd`.

## Templates

| Shortcut       | Action                       |
| -------------- | ---------------------------- |
| `Ctrl-Shift-L` | Create a new lecture note    |
| `Ctrl-Shift-A` | Create a new assignment note |
| `Ctrl-Shift-T` | Create a new topic note      |

## Navigation

| Shortcut | Action                   |
| -------- | ------------------------ |
| `Ctrl-F` | Search the entire vault. |
